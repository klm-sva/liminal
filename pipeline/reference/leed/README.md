# LEED Reference Files

Place these four files here before running the pipeline:

| File | Description |
|------|-------------|
| `LEED_v41_BDC_Automation_Analysis_v9.xlsx` | Authoritative automation analysis spreadsheet — single source of truth for all credit columns |
| `leed_v41_form_schemas.json` | Live LEED Online form field schemas for all 53 LEED v4.1 BD+C credits (generated by scrape_leed_forms.js) |
| `leed_v41_calculator_schemas.json` | USGBC calculator field schemas (generated by parse_leed_calculators.js) |
| `LEED_DATABASE_README.md` | Reference database documentation |

**Re-scrape:** Run `node scrape_leed_forms.js` if USGBC updates their forms.
